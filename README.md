# 📌 Overview

The **Global Development AI Intelligence Platform** is a premium, highly interactive, AI-powered analytics system developed using **Streamlit**, **Machine Learning**, and **Advanced Data Visualization** techniques.

This project performs:

- 🌍 Global country clustering
- 📊 Economic intelligence analysis
- 🤖 AI-generated recommendations
- 📈 Comparative clustering model evaluation
- 🧠 Real-time clustering visualization
- 🗺️ Interactive global mapping
- 🔍 Exploratory Data Analysis (EDA)
- ⚡ Advanced AI dashboard analytics

The platform transforms raw development metrics into intelligent, interactive insights.

---

# 🎯 Business Objective

The objective of this project is to:

- Analyze global development indicators
- Identify hidden country clusters
- Compare clustering algorithms
- Generate AI-powered development insights
- Visualize economic intelligence patterns
- Deploy a production-ready analytics platform

# 🧠 Machine Learning Algorithms Used

The platform supports multiple clustering models:

| Algorithm | Purpose |
|---|---|
| KMeans | Partition-based clustering |
| DBSCAN | Density-based clustering |
| Agglomerative Clustering | Hierarchical clustering |
| Gaussian Mixture Models | Probabilistic clustering |
| Spectral Clustering | Graph-based clustering |
| Mean Shift | Centroid-based clustering |

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/7-vsd/Global_Development_AI_Platform.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd Global_Development_AI_Platform
```

---

## 3️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

```bash
streamlit run app.py
```

Open browser:

```bash
http://localhost:8501
```
